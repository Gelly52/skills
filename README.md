# Skills

Reusable agent skills for AI coding assistants.

## Install

```bash
npx skills add Gelly52/skills
```

Or install a specific skill:

```bash
npx skills add Gelly52/skills --skill=YoN
```

## Skills

| Skill | Description |
|-------|-------------|
| [YoN](./YoN/) | Stress-test a plan or design by reducing every decision to Yes/No/Custom, one at a time. Produces a `DECISIONS.md` conclusions document. |
